## ðŸŽ“ Learning Objectives

This workshop project teaches the following SwiftUI concepts:

### Core SwiftUI Concepts
- **Views & Layouts**: `VStack`, `HStack`, `ZStack`, `ScrollView`
- **State Management**: `@State`, `@AppStorage`, `@Environment`
- **Navigation**: `NavigationStack`, `NavigationLink`, `navigationDestination`
- **View Modifiers**: Styling, typography, colors, and layout modifiers
- **Reusable Components**: Building custom view components

### Data & Persistence
- **SwiftData**: `@Model`, `@Query`, `@Relationship`
- **Model Container**: Setup and configuration
- **Data Relationships**: One-to-many relationships with cascade delete

### Advanced Topics
- **Custom Extensions**: Extending `Color` for hex conversion
- **Computed Properties**: Dynamic UI properties based on data
- **Date Formatting**: Working with `DateFormatter`
- **SF Symbols**: Using system icons
- **Animations**: Spring and ease animations

### iOS Architecture
- **App Lifecycle**: `@main`, `App` protocol
- **Scene Management**: `WindowGroup` and conditional rendering
- **Permissions**: User authorization patterns (planned for Speech framework)
